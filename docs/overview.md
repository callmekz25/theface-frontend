# Skincare Microservices Frontend

## Project Overview

This project is the frontend for a skincare e-commerce microservices system.

The frontend communicates with backend microservices through an API Gateway.

Main features:

- Product browsing
- Product variants
- Cart management
- User authentication
- Checkout

---

## Tech Stack

Frontend:

- React
- TypeScript
- Vite
- React Router
- React Query
- Axios

Backend:

- Spring Boot Microservices
- API Gateway
- PostgreSQL
